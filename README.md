# Azur Lane Ship Images

Extracted ship paintings and thumbnails for use by [AzurAPI](https://github.com/iujab/AzurAPI).

- `paintings/` — Skin art in WebP format, named by internal asset name (e.g. `beierfasite.webp` = Belfast default)
- `thumbnails/` — Shipyard icons in PNG format, named by group_type ID (e.g. `20212.png` = Belfast)

Images extracted from the EN game client using [AzurLane-AssetDownloader](https://github.com/nobbyfix/AzurLane-AssetDownloader).
